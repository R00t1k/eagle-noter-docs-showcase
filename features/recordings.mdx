## Übersicht

Der Bereich **Aufnahmen** in Eagle Noter ermöglicht es Ihnen, eigene Audio- und Videodateien zur Transkription und KI-Analyse hochzuladen. Egal ob Sie eine Aufnahme aus einer externen Quelle haben oder ein Meeting verarbeiten möchten, das nicht von unserem Bot erfasst wurde, können Sie es hier hochladen.

## Aufnahmen hochladen

### Unterstützte Formate

Eagle Noter unterstützt folgende Audio- und Videoformate:

**Audio:** MP3, M4A, WAV, OGG, FLAC, AAC, OPUS  
**Video:** MP4, WEBM

**Maximale Dateigröße:** 500 MB pro Upload

### So laden Sie hoch

<Steps>
  <Step title="Gehen Sie zu Aufnahmen">
    Navigieren Sie zum Tab **Aufnahmen** in Ihrem Arbeitsbereichs-Dashboard.
  </Step>
  <Step title="Klicken Sie auf Hochladen">
    Klicken Sie auf die Schaltfläche **Aufnahme hochladen** in der oberen
    rechten Ecke.
  </Step>
  <Step title="Wählen Sie Upload-Methode">
    Sie können entweder: - **Eine Datei hochladen** direkt von Ihrem Computer -
    **Eine URL einfügen** zu einer öffentlich zugänglichen Audio-/Videodatei
  </Step>
  <Step title="Details hinzufügen">
    - **Titel**: Geben Sie Ihrer Aufnahme einen beschreibenden Namen -
    **Beschreibung** (optional): Fügen Sie Notizen zur Aufnahme hinzu -
    **Sprache**: Wählen Sie die in der Aufnahme gesprochene Sprache
  </Step>
  <Step title="Verarbeiten">
    Klicken Sie auf **Hochladen** und warten Sie auf die Verarbeitung. Sie sehen
    den Echtzeit-Fortschritt, während Ihre Datei hochgeladen und transkribiert
    wird.
  </Step>
</Steps>

### Upload von URL

Haben Sie eine online gehostete Aufnahme? Fügen Sie einfach die direkte URL zur Audio- oder Videodatei ein, und Eagle Noter lädt sie automatisch herunter und verarbeitet sie.

<Note>
  Die URL muss ein direkter Link zur Mediendatei sein (endend auf .mp3, .mp4,
  etc.) oder ein öffentlich zugänglicher Speicherlink.
</Note>

## Aufnahmen ansehen

Sobald verarbeitet, erscheinen Ihre Aufnahmen in der Aufnahmen-Liste. Klicken Sie auf eine Aufnahme, um zu sehen:

- **Audio-Player**: Spielen Sie die Originalaufnahme direkt in Ihrem Browser ab
- **Vollständiges Transkript**: Lesen Sie die vollständige Transkription mit Zeitstempeln
- **KI-Zusammenfassung**: Von der KI extrahierte Kernpunkte, Entscheidungen und Aufgaben
- **Export-Optionen**: Laden Sie Transkripte als Untertitel (SRT/VTT) oder Text herunter

## Aufnahmen verwalten

### Wiedergabe

Jede Aufnahme enthält einen eingebauten Audio-Player, mit dem Sie:

- Abspielen, pausieren und durch die Aufnahme suchen können
- Die Wiedergabegeschwindigkeit anpassen
- Zu im Transkript erwähnten spezifischen Zeitstempeln springen

### Massenauswahl

Sie können mehrere Aufnahmen mit den Kontrollkästchen auswählen, um Massenaktionen wie Löschen durchzuführen.

### Aufnahmen löschen

So löschen Sie eine Aufnahme:

1. Wählen Sie eine oder mehrere Aufnahmen mit den Kontrollkästchen aus
2. Klicken Sie auf die Schaltfläche **Löschen**
3. Bestätigen Sie die Löschung

<Warning>
  Das Löschen einer Aufnahme entfernt dauerhaft die Audiodatei, das Transkript
  und die KI-Zusammenfassung. Diese Aktion kann nicht rückgängig gemacht werden.
</Warning>

## Sprachunterstützung

Wählen Sie beim Hochladen die Transkriptionssprache:

| Sprache                  | Code   | Beschreibung                                         |
| ------------------------ | ------ | ---------------------------------------------------- |
| Auto (Mehrsprachig)      | `auto` | Erkennt automatisch die gesprochene Sprache          |
| Englisch                 | `en`   | Am besten für englische Aufnahmen                    |
| Deutsch/Schweizerdeutsch | `de`   | Optimiert für Deutsch und Schweizerdeutsche Dialekte |
| Französisch              | `fr`   | Für französischsprachige Aufnahmen                   |

<Tip>
  Verwenden Sie **Auto (Mehrsprachig)**, wenn Ihre Aufnahme mehrere Sprachen
  enthält oder Sie sich über die Hauptsprache unsicher sind.
</Tip>

## Tipps für beste Ergebnisse

<AccordionGroup>
  <Accordion title="Audioqualität">
    Klares Audio mit minimalen Hintergrundgeräuschen produziert die besten
    Transkriptionen. Verwenden Sie wenn möglich Aufnahmen von dedizierten
    Mikrofonen statt Laptop-Lautsprechern.
  </Accordion>
  <Accordion title="Sprecherklarheit">
    Stellen Sie sicher, dass Sprecher nicht übereinander sprechen. Klares
    Abwechseln hilft bei genauer Transkription und Sprechererkennung.
  </Accordion>
  <Accordion title="Dateigröße">
    Bei sehr langen Aufnahmen (über 2 Stunden) erwägen Sie, sie in kleinere
    Segmente aufzuteilen für schnellere Verarbeitung.
  </Accordion>
</AccordionGroup>
